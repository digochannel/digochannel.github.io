Guia completo — hospedar em 30 minutos
⚠️ Antes de começar: você precisa substituir os valores COLE_AQUI_… no início deste arquivo HTML pelas credenciais do seu Firebase. Siga os passos abaixo.
1
Criar conta no Firebase (gratuita)
Acesse firebase.google.com → clique em Get started → entre com sua conta Google.
2
Criar um novo projeto
Clique em Add project → dê um nome (ex: intimacoes-2tab) → desative o Google Analytics → clique em Create project.
3
Ativar o Realtime Database
No menu lateral: Build → Realtime Database → Create database.
Escolha a região us-central1 → selecione Start in test mode → clique em Enable.
4
Copiar as credenciais
No menu lateral: Project settings (ícone de engrenagem) → aba General → role até "Your apps" → clique em </> (Web) → dê um apelido → clique em Register app.
Vai aparecer um bloco firebaseConfig = { ... }. Copie todos os valores e cole no início deste arquivo HTML substituindo os COLE_AQUI_….
5
Criar conta no GitHub e hospedar
1. Acesse github.com → crie uma conta gratuita.
2. Clique em New repository → nome: intimacoes → marque Public → Create repository.
3. Clique em uploading an existing file → arraste o arquivo HTML → Commit changes.
4. Vá em Settings → Pages → Branch: main → Save.
5. Em ~1 minuto o sistema estará disponível em:
https://SEU_USUARIO.github.io/intimacoes/sistema_intimacoes_online.html
6
Autorizar o domínio no Firebase
Firebase console → Realtime Database → Rules → substitua por:
{ "rules": { ".read": true, ".write": true } }
→ Publish. (Para o teste inicial — depois configuramos login com senha.)
7
Instalar como app
PC (Chrome/Edge): abra a URL → clique no ícone ⊕ na barra de endereços → Instalar.
Android: Menu ⋮ → Adicionar à tela inicial.
iPhone: Compartilhar ⎙ → Adicionar à tela de início.
✅ Pronto! Qualquer pessoa que abrir o link verá os dados em tempo real. O motoboy abre pelo celular, o supervisor abre no computador — ambos sincronizados automaticamente.
